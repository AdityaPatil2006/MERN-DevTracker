# DevTracker

## Features

- Add Project
- View Project
- Edit Project
- Delete Project
- Filter Projects by Status
- Search Projects
- Store Github Repository Links
- Store Deployment Links

## Project Status

- Planning
- In Progress
- Completed

## Technologies

- React
- Node.js
- Express.js
- MongoDB

## User-Inputs

- Add Project
  Project Name
  Project Description
  Tech Stack
  Status
  Github Repository Link
  Deployment Link
  CreatedAt

## API Routes

router.get("/") -> Get All Projects
router.get("/:id") -> Get Specific Projects
router.post("/") -> Add Project
router.put("/:id") -> Update Project
router.delete("/:id") -> Delete Project
